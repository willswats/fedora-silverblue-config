# distrobox

## Create

```bash
distrobox create --name main --home ~/Local/Distrobox/main
```

## Update

```bash
sudo dnf update
```

## Install

```bash
sudo dnf install fish
```

### nvm & node

Install nvm:

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

Install latest node:

```bash
nvm install node
```
